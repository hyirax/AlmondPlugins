### ✨ 概要

**AlmondHousing**は、[DisPlace](https://github.com/Drakansoul/DisPlace)をベースに大幅に再構築・最適化されたFF14ハウジング支援プラグインです。**AlmondCookie**によって独立してメンテナンスされています。複雑なハウジング、素材の管理、データ共有をよりオタクらしく、純粋で簡単にすることを目指しています。

**主な機能：**
* 🌍 **多言語シームレス切り替え**：日本語、英語、中国語（簡体/繁体）、韓国語、ドイツ語、フランス語をネイティブサポート。
* 📦 **スマート材料チェック**：所持品とチョコボかばんを自動的にスキャンし、現在のレイアウトと比較して、不足している家具を正確に計算します。
* 📊 **高度なリスト出力**：所持済みの家具を除外した**Teamcraft用購入リスト**、または詳細なデータを含むCSVファイルをワンクリックで出力できます。
* 🏗️ **図面の互換性と保護**：MakePlaceおよびReMakePlaceの`.json`図面の読み込みと完全な下位互換性を持ちます。また、不正な改ざんや盗用を防ぐため、ChaCha20-Poly1305アルゴリズムを採用した専用の`.almond`暗号化フォーマットでの保存もサポートしています。

### 🚀 インストール方法

1. **前提条件**：[Dalamud / FFXIV Quick Launcher](https://goatcorp.github.io/)を使用してゲームを起動していること。
2. ゲーム内のチャットで `/xlsettings` と入力し、Dalamudの設定を開きます。
3. **「試験的機能 (Experimental)」**タブに切り替えます。
4. **「カスタムプラグインリポジトリ」**に以下のリンクを追加します：
   `https://raw.githubusercontent.com/hyirax/AlmondHousing/master/pluginmaster.json`
5. **+** ボタンをクリックし、**「有効 (Enabled)」**にチェックが入っていることを確認して、「保存して閉じる」をクリックします。
6. `/xlplugins` のプラグインインストーラーで **`AlmondHousing`** を検索してインストールします。

---

### 🎮 クイックガイド

チャットで **`/almond`** と入力すると、コントロールパネルが開きます。

1. **レイアウトの保存**：権限を持つハウスに入り、システム標準の「調度品を回転する」モードをオンにした後、`名前を付けて保存 (Save As)` をクリックすると、現在のハウスデータをバックアップできます。
2. **材料の準備**：`材料チェック (Material Audit)` タブをクリックして不足している家具を確認し、リストを出力して正確に買い出しを行います。
3. **レイアウトの適用**：図面に必要な家具を所持品に入れるか、床に配置した後、`ファイルを選択して適用 (Select File & Apply)` をクリックすると、プラグインが自動的に配置を完了します。

*(注意：ゲームのクラッシュを防ぐため、自動配置中はマウスを動かしたり、キャラクターが家具の移動経路を塞いだりしないようにしてください。)*
