### ✨ 概要

**AlmondHousing** は、FF14のハウジング支援ツールです。コミュニティの先人たちのプロジェクトの探求と蓄積から生まれ、**AlmondCookie** が独立してメンテナンスしています。ゲームの許容範囲内で、必要とするプレイヤーにいくつかの追加的な利便性を提供することを目指しています。

**主な機能:**
* 🌍 **多言語サポート** — 中国語（簡体/繁体）、英語、日本語、韓国語、ドイツ語、フランス語にネイティブ対応
* 📦 **素材チェック** — 所持品とチョコボかばんを自動スキャンし、現在の図面とリアルタイムで比較、不足分を正確に計算
* 📊 **リスト出力** — Teamcraft用購入リストやCSV表をワンクリックでエクスポート
* 🔐 **図面互換性** — MakePlace、ReMakePlaceの `.json` 図面を読み取り可能、`.almond` 形式での保存に対応

### 🚀 インストール

1. [Dalamud / FFXIV Quick Launcher](https://goatcorp.github.io/) を使用してゲームを起動
2. チャットで `/xlsettings` と入力し、Dalamud設定を開く
3. **「試験的機能」** タブに切り替え
4. **「カスタムプラグインリポジトリ」** に以下を追加:
   `https://raw.githubusercontent.com/hyirax/AlmondHousing/master/pluginmaster.json`
5. **+** をクリックし、**「有効」** にチェックを入れて保存して閉じる
6. `/xlplugins` で **AlmondHousing** を検索してインストール

---

### 📋 クイックスタート

チャットで **`/almond`** と入力してパネルを開きます。

1. **レイアウト保存** — 権限のあるハウスに入り、ハウジング回転モードをオンにして `名前を付けて保存` をクリック
2. **素材準備** — `素材チェック` タブで不足アイテムを確認し、リストをエクスポートして正確に購入
3. **レイアウト適用** — 必要な家具を所持品に入れるか床に置き、`ファイルを選択して適用` をクリックで自動配置

*(自動配置中は、マウスを動かしたりキャラクターが家具の移動経路を塞がないようご注意ください)*
