## AlmondHousingについて

**AlmondHousing**は、**AlmondCookie**が独自に設計・保守しているFFXIVハウジング支援ツールです。家具の閲覧、レイアウト管理、素材確認、レイアウト共有を一つの見やすい画面にまとめています。

## 無料利用について

**プラグイン本体は無料です。転売や有料ダウンロードでの配布は禁止します。** 有料で入手した場合は、返金を求めて販売者を報告してください。

ハウジングレイアウトには他者の創作物が含まれる場合があります。共有・取引できる権利を持つデータだけを扱い、元のデザイナーの許諾条件を守ってください。

## 機能

- **家具と内装部品**：室内家具、庭具、内装部品をリアルタイムで検索・分類・絞り込み。
- **レイアウト管理**：ハウスや庭のレイアウトを読み込み、照合、適用、出力。出力に模様替え権限は不要です。
- **素材確認**：所持品とチョコボかばんを走査し、必要数・所持数・不足数を比較してTeamcraftまたはCSVリストを出力。
- **レイアウト互換**：MakePlaceとReMakePlaceの`.json`を読み込み、共有データを`.almond`で保存。
- **保護された共有**：標準暗号化または任意のパスワードを使用。保護されたファイルの読み込み時は自動でパスワード入力を表示します。
- **精密調整**：座標ドラッグ、スナップ、グループ移動、元に戻す/やり直し、3D操作軸。

## インストール

1. [Dalamud / FFXIV Quick Launcher](https://goatcorp.github.io/)でゲームを起動します。
2. チャットに`/xlsettings`と入力してDalamud設定を開きます。
3. **試験的機能 (Experimental)** の **カスタムプラグインリポジトリ** に次のURLを追加します。

   `https://raw.githubusercontent.com/hyirax/AlmondPlugins/main/pluginmaster.json`

4. リポジトリを有効にして設定を保存します。
5. `/xlplugins`で**AlmondHousing**を検索してインストールします。

## クイックスタート

1. `/almond`と入力してプラグインを開きます。
2. 読み取り可能なハウスまたは庭に入り、**レイアウトと出力**から**名前を付けて保存**を選びます。
3. **素材確認**で不足品を確認し、必要に応じて買い物リストを出力します。
4. レイアウト適用前に必要な家具を現在のハウス内へ配置し、模様替えの回転モードを有効にして**ファイルから適用**を選びます。

移動されるのは現在のハウスに配置済みの家具だけです。適用中は回転モードを維持し、家具を手動で動かしたりハウスから退出したりしないでください。

## レイアウトとパスワード

- パスワードを空欄にすると、`.almond`は標準暗号化を使用します。
- 任意のパスワードは6文字以上必要で、現在のプラグインセッション内だけに保持されます。
- パスワード付き`.almond`を読み込むと入力画面が開きます。通常のファイルでは表示されません。
