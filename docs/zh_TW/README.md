### ✨ 簡介

**AlmondHousing** 是一款專注於 FF14 房屋裝修場景的增強工具，脫胎於社群前輩專案的探索與積累，由 **AlmondCookie** 獨立維護。希望在遊戲允許的框架內，為有需求的玩家提供一些額外的便利。

**主要功能：**
* 🌍 **多語言支援** — 原生支援中文（簡/繁）、英語、日語、韓語、德語、法語
* 📦 **物料盤點** — 自動掃描背包與陸行鳥鞍囊，即時對比當前圖紙，精準計算缺件
* 📊 **清單匯出** — 一鍵匯出 Teamcraft 採購單或 CSV 表格
* 🔐 **圖紙相容** — 可讀取 MakePlace、ReMakePlace 的 `.json` 圖紙，支援儲存為 `.almond` 加密格式

### 🚀 安裝

1. 使用 [Dalamud / FFXIV Quick Launcher](https://goatcorp.github.io/) 啟動遊戲
2. 在聊天框輸入 `/xlsettings` 開啟衛月設定
3. 切換到 **「實驗性功能」** 標籤頁
4. 在 **「自訂外掛倉庫」** 中新增：
   `https://raw.githubusercontent.com/hyirax/AlmondHousing/master/pluginmaster.json`
5. 點選 **+** 並勾選 **「啟用」**，然後儲存並關閉
6. 在 `/xlplugins` 中搜尋 **AlmondHousing** 安裝

---

### 📋 快速上手

聊天框輸入 **`/almond`** 開啟面板。

1. **儲存佈局** — 進入有權限的房屋，開啟裝修旋轉模式，點選 `另存新檔` 備份
2. **物料準備** — 切換到 `物料盤點` 檢視缺件，匯出清單精準採購
3. **套用佈局** — 將所需家具放入背包或放置地面，點選 `選擇檔案並套用` 自動擺放

*(自動擺放期間請勿移動滑鼠或讓人物擋住家具移動路線)*
