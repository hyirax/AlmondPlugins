## 其略

**AlmondHousing** 者，**AlmondCookie** 所獨創而自持也。合察器、治圖、稽料、傳圖諸事於一牖，使宅居之務簡而有序。

## 無直而用

**此器無直，毋售其器，亦毋匿於有償之門。** 若自市肆得之，宜索還其直而告其所。

屋圖或載他人心力；非有其權，毋擅出、傳、售，當循原作者所許。

## 所具之能

- **察器**：即時索引、分類、簡擇室中、庭中與屋體諸器。
- **治圖**：讀取、相合、施用、匯出屋宇或庭院之圖；匯出不須營造之權。
- **稽料**：察行囊與陸行鳥鞍囊，校所需、所有、所闕，並出 Teamcraft 或 CSV 清冊。
- **通圖**：可讀 MakePlace、ReMakePlace 之 `.json`，亦以 `.almond` 存其圖數。
- **護傳**：可循常法加密，亦可自定密語；入有密語之圖，器自請其密。
- **微調**：可曳坐標、吸附、群移、復前行、重行，亦有三維操軸。

## 安置

1. 先以 [Dalamud / FFXIV Quick Launcher](https://goatcorp.github.io/) 啟戲。
2. 於遊戲談欄書 `/xlsettings`，啟 Dalamud 之設。
3. 入 **Experimental**，於 **Custom Plugin Repositories** 添：

   `https://raw.githubusercontent.com/hyirax/AlmondPlugins/main/pluginmaster.json`

4. 啟其庫而存之。
5. 於 `/xlplugins` 索 **AlmondHousing** 而安之。

## 初用

1. 書 `/almond` 以啟其牖。
2. 入可讀之屋或庭，於 **佈局與匯出** 擇 **另存為**。
3. 入 **物料盤點**，察所闕之器，按需出採買清冊。
4. 欲施圖，先置所需家具於當前屋中，啟裝修旋轉模式，乃擇 **從檔案套用**。

此器惟相合而移屋中已置之家具。施圖之時，當常守旋轉模式，毋手移家具，毋離其屋。

## 圖與密語

- 密語若空，`.almond` 用常法加密。
- 自定密語至少六字，且僅存於當次會話。
- 入有密語之 `.almond`，器將啟牖求密；常圖則不問。
