### ✨ 소개

**AlmondHousing**은 [DisPlace](https://github.com/Drakansoul/DisPlace)를 기반으로 깊이 있게 재구축되고 최적화된 FF14 하우징 보조 플러그인으로, **AlmondCookie**가 독립적으로 유지보수하고 있습니다. 복잡한 하우징, 재료 관리, 데이터 공유를 더 긱(geek)스럽고 순수하며 단순하게 만드는 데 전념하고 있습니다.

**핵심 기능:**
* 🌍 **다국어 원활한 전환**: 한국어, 영어, 중국어(간체/번체), 일본어, 독일어, 프랑스어를 기본 지원하여 글로벌 및 한국 서버의 모든 환경을 충족합니다.
* 📦 **스마트 재료 확인**: 플레이어의 소지품과 초코보 가방을 자동으로 스캔하여 현재 도면과 실시간으로 비교하고, 가구의 "부족분"을 정확하게 계산합니다.
* 📊 **고급 목록 내보내기**: 이미 보유한 가구를 제외한 **Teamcraft 구매 목록**이나 상세한 잉여/부족 데이터가 포함된 CSV 표를 원클릭으로 내보냅니다.
* 🏗️ **도면 호환성 및 보호**: MakePlace 및 ReMakePlace의 `.json` 도면을 완벽하게 하위 호환하여 읽을 수 있으며, ChaCha20-Poly1305 알고리즘을 사용한 전용 `.almond` 암호화 형식으로 저장하여 무단 변조 및 도용을 방지합니다.

### 🚀 설치 방법

1. **전제 조건**: [Dalamud / FFXIV Quick Launcher](https://goatcorp.github.io/)를 사용하여 게임을 실행하십시오.
2. 게임 채팅창에 `/xlsettings`를 입력하여 Dalamud 설정을 엽니다.
3. **"실험적 기능 (Experimental)"** 탭으로 전환합니다.
4. **"사용자 정의 플러그인 리포지토리"**에 다음 링크를 추가합니다:
   `https://raw.githubusercontent.com/hyirax/AlmondHousing/master/pluginmaster.json`
5. **+** 버튼을 클릭하고 **"활성화 (Enabled)"**를 선택한 후 "저장 및 닫기"를 클릭합니다.
6. `/xlplugins` 플러그인 설치기에서 **`AlmondHousing`**을 검색하여 설치합니다.

### 🎮 빠른 시작 가이드

채팅창에 **`/almond`**를 입력하여 제어판을 엽니다.

1. **레이아웃 저장**: 권한이 있는 하우징에 들어가서 기본 "가구 회전(Rotate Furniture)" 모드를 켠 후, `다른 이름으로 저장 (Save As)`을 클릭하여 현재 하우징 데이터를 백업합니다.
2. **재료 준비**: `재료 확인 (Material Audit)` 탭을 클릭하여 부족한 가구를 확인하고 목록을 내보내어 정확하게 구매합니다.
3. **레이아웃 적용**: 도면에 필요한 가구를 소지품에 넣거나 바닥에 배치한 후, `파일 선택 및 적용 (Select File & Apply)`을 클릭하면 플러그인이 자동으로 배치를 완료합니다.

*(참고: 게임 크래시를 방지하기 위해 자동 배치 중에는 마우스를 움직이거나 캐릭터가 가구 이동 경로를 막지 않도록 하십시오.)*
