이 도구의 목적은 플레이어가 자신의 하우징 레이아웃을 백업하고 무료 도면을 커뮤니티에서 공유할 수 있도록 돕는 것입니다. 커뮤니티의 성과가 이익을 위해 악용되어서는 안 됩니다. 유료 채널을 통해 이 도구를 입수하셨다면 환불을 요청하고 판매자를 신고하시기 바랍니다.

### ✨ 소개

**AlmondHousing**은 FF14 하우징 지원 도구로, 커뮤니티 선배 프로젝트들의 탐구와 축적에서 비롯되었으며 **AlmondCookie**가 독립적으로 유지보수하고 있습니다. 게임이 허용하는 범위 내에서 필요한 플레이어에게 몇 가지 추가적인 편의를 제공하는 것을 목표로 합니다.

**주요 기능:**
* 🌍 **다국어 지원** — 중국어(간체/번체), 영어, 일본어, 한국어, 독일어, 프랑스어 기본 지원
* 📦 **자재 점검** — 소지품과 초코보 가방을 자동 스캔하여 현재 도면과 실시간 비교, 부족분을 정확히 계산
* 📊 **목록 내보내기** — Teamcraft 구매 목록 또는 CSV 표로 원클릭 내보내기
* 🔐 **도면 호환성** — MakePlace, ReMakePlace의 `.json` 도면 읽기 가능, `.almond` 형식 저장 지원

### 🚀 설치

1. [Dalamud / FFXIV Quick Launcher](https://goatcorp.github.io/)를 사용하여 게임 실행
2. 채팅에 `/xlsettings` 입력하여 Dalamud 설정 열기
3. **"실험적 기능"** 탭으로 전환
4. **"사용자 정의 플러그인 리포지토리"** 에 다음 추가:
   `https://raw.githubusercontent.com/hyirax/AlmondHousing/master/pluginmaster.json`
5. **+** 클릭 후 **"활성화"** 체크하고 저장 후 닫기
6. `/xlplugins`에서 **AlmondHousing** 검색하여 설치

---

### 📋 빠른 시작

채팅에 **`/almond`** 입력하여 패널을 엽니다.

1. **레이아웃 저장** — 권한이 있는 주택에 들어가 가구 회전 모드를 켜고 `다른 이름으로 저장` 클릭
2. **자재 준비** — `자재 점검` 탭에서 부족한 항목을 확인하고 목록을 내보내 정확히 구매
3. **레이아웃 적용** — 필요한 가구를 소지품이나 바닥에 놓고 `파일 선택 및 적용` 클릭으로 자동 배치

*(자동 배치 중에는 마우스를 움직이거나 캐릭터가 가구 이동 경로를 막지 않도록 주의하세요)*

### 🙏 감사의 말

이 프로젝트는 커뮤니티의 많은 선구자들의 탐구와 축적 없이는 불가능했습니다:

* **Drakansoul** — DisPlace 제작 및 유지보수
* **NotNite** — 적절한 가이드라인에 따라 MakePlace 플러그인 코드의 공개를 촉진
* **jawslouis** — 뛰어난 MakePlacePlugin 제작
* **HousingPos**, **BDTH**, **HouseMate** 등 기술적 선배들의 영감에 감사드립니다
