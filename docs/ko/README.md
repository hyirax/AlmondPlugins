## 소개

**AlmondHousing**은 **AlmondCookie**가 독립적으로 설계하고 유지보수하는 FFXIV 하우징 도구입니다. 가구 탐색, 레이아웃 관리, 재료 점검 및 레이아웃 공유를 하나의 명확한 화면에 모았습니다.

## 무료 이용

**플러그인 자체는 무료입니다. 재판매하거나 유료 다운로드로 배포하지 마십시오.** 유료로 받았다면 환불을 요청하고 판매자를 신고해 주십시오.

하우징 레이아웃에는 다른 사람의 창작물이 포함될 수 있습니다. 사용 권한이 있는 데이터만 공유하거나 거래하고 원작 디자이너의 허가 조건을 지켜 주십시오.

## 기능

- **가구와 내장재**: 실내 가구, 정원 가구 및 내장재를 실시간으로 검색, 분류 및 필터링합니다.
- **레이아웃 관리**: 하우스와 정원의 레이아웃을 읽고, 매칭하고, 적용하고, 내보냅니다. 내보내기에는 꾸미기 권한이 필요하지 않습니다.
- **재료 점검**: 소지품과 초코보 가방을 스캔해 필요, 보유, 부족 수량을 비교하고 Teamcraft 또는 CSV 목록을 내보냅니다.
- **레이아웃 호환**: MakePlace와 ReMakePlace의 `.json` 파일을 읽고 공유 데이터를 `.almond`로 저장합니다.
- **보호된 공유**: 표준 암호화 또는 선택적 사용자 지정 비밀번호를 사용합니다. 보호된 파일을 불러오면 비밀번호 입력창이 자동으로 열립니다.
- **정밀 조정**: 좌표 드래그, 스냅, 그룹 이동, 실행 취소/다시 실행 및 3D 조작축을 제공합니다.

## 설치

1. [Dalamud / FFXIV Quick Launcher](https://goatcorp.github.io/)로 게임을 실행합니다.
2. 채팅에 `/xlsettings`를 입력해 Dalamud 설정을 엽니다.
3. **실험적 기능 (Experimental)** 의 **사용자 지정 플러그인 저장소**에 다음 URL을 추가합니다.

   `https://raw.githubusercontent.com/hyirax/AlmondPlugins/main/pluginmaster.json`

4. 저장소를 활성화하고 설정을 저장합니다.
5. `/xlplugins`에서 **AlmondHousing**을 검색해 설치합니다.

## 빠른 시작

1. `/almond`를 입력해 플러그인을 엽니다.
2. 읽을 수 있는 하우스나 정원에 들어가 **레이아웃 및 내보내기**에서 **다른 이름으로 저장**을 선택합니다.
3. **재료 점검**에서 부족한 물품을 확인하고 필요할 때 구매 목록을 내보냅니다.
4. 레이아웃 적용 전에 필요한 가구를 현재 하우스에 배치하고 가구 회전 모드를 켠 뒤 **파일에서 적용**을 선택합니다.

플러그인은 현재 하우스에 이미 배치된 가구만 매칭하고 이동합니다. 적용 중에는 회전 모드를 유지하고 가구를 직접 움직이거나 하우스를 떠나지 마십시오.

## 레이아웃과 비밀번호

- 비밀번호를 비워 두면 `.almond`는 표준 암호화를 사용합니다.
- 사용자 지정 비밀번호는 6자 이상이어야 하며 현재 플러그인 세션에만 보관됩니다.
- 비밀번호가 있는 `.almond`를 불러오면 입력창이 열립니다. 일반 파일에는 표시되지 않습니다.
