### ✨ 简介

**AlmondHousing** 是一款专注于 FF14 房屋装修场景的增强工具，脱胎于社区前辈项目的探索与积累，由 **AlmondCookie** 独立维护。希望在游戏允许的框架内，为有需求的玩家提供一些额外的便利。

**主要功能：**
* 🌍 **多语言支持** — 原生支持中文（简/繁）、英语、日语、韩语、德语、法语
* 📦 **物料盘点** — 自动扫描背包与陆行鸟鞍囊，实时对比当前图纸，精准计算缺件
* 📊 **清单导出** — 一键导出 Teamcraft 采购单或 CSV 表格
* 🔐 **图纸兼容** — 可读取 MakePlace、ReMakePlace 的 `.json` 图纸，支持保存为 `.almond` 加密格式

### 🚀 安装

1. 使用 [Dalamud / FFXIV Quick Launcher](https://goatcorp.github.io/) 启动游戏
2. 在聊天框输入 `/xlsettings` 打开卫月设置
3. 切换到 **"实验性功能"** 标签页
4. 在 **"自定义插件仓库"** 中添加：
   `https://raw.githubusercontent.com/hyirax/AlmondHousing/master/pluginmaster.json`
5. 点击 **+** 并勾选 **"启用"**，然后保存并关闭
6. 在 `/xlplugins` 中搜索 **AlmondHousing** 安装

---

### 📋 快速上手

聊天框输入 **`/almond`** 打开面板。

1. **保存布局** — 进入有权限的房屋，开启装修旋转模式，点击 `另存为` 备份
2. **物料准备** — 切换到 `物料盘点` 查看缺件，导出清单精准采购
3. **应用布局** — 将所需家具放入背包或放置地面，点击 `选择文件并应用` 自动摆放

*(自动摆放期间请勿移动鼠标或让人物挡住家具移动路线)*
