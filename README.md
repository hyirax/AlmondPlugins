# Almond Plugins

AlmondCookie 的 FFXIV Dalamud 插件发布仓库。此仓库仅提供插件目录、说明与编译后的发布包。

## 安装

在 Dalamud 设置的 **实验性功能 > 自定义插件仓库** 中添加：

```text
https://raw.githubusercontent.com/hyirax/AlmondPlugins/main/pluginmaster.json
```

保存后，在 `/xlplugins` 中搜索插件名称即可安装或更新。

## 插件

| 插件 | 命令 | 当前版本 | 简介 |
| --- | --- | --- | --- |
| AlmondHousing · 杏仁造境 | `/almond` | 1.1.4 | 家具检索、布局管理、物料盘点、加密分享与精细摆放 |
